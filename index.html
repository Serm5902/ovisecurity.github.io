Seguridad Informática

La seguridad informática es un punto esencial para las tecnologías de la información permite proteger datos de las organizaciones ante cualquier calamidad como pueden ser desastres naturales o ataques informáticos.

Para llevar acabo todo este proceso es necesario conocer los modelos y estándares en seguridad informática para poder tener una buena protección en los sistemas, por tal motivo se crea este objeto virtual interactivo por sus siglas O.V.I que tiene como objetivos:

  1. Conocer los modelos en la seguridad informática y que normas los rigen de una manera corta.
  2. Comprender los diferentes estándares de seguridad informática en un sistema.
  3. Conocer las normas que minimizan los riesgos a la información o infraestructura informática.
  4. Conocer y aplicar los diferentes planes que se ejecutan en la seguridad informática.

Las temáticas a tratar en este curso son las siguientes:

  1. Plan de continuidad del negocio
  2. Plan de respuesta a incidentes
  3. Gobierno TI
  4. Cobit
  5. ITIL
  6. Normas para la protección de activos

Aparte de estas temáticas se estarán presentando dos videos acerca de Gobierno Ti y ITIL para poder explicar lo mejor posible estos dos temas de gran importancia. También se cuenta con tres (3) actividades para reforzar lo que el estudiante haya aprendido durante el proceso de aprendizaje de este objeto virtual de aprendizaje. Al costa derecho de la pagina encontrara un acceso rápido a las temáticas que manejara este O.V.I mencionadas anteriormente pero igual mente podrá acceder desde el menú de navegación en la parte superior.
